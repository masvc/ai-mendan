# 応募者向け（面接を受ける方の画面）

応募者がスマートフォンからアクセスした際の画面です。

## 面接の流れ

### 1. トップ画面

スマートフォンで上記URLを開くと、トップ画面が表示されます。

- **「初めから」**ボタンを押すと面接が始まります
- 途中で中断した場合は**「続きから」**ボタンで再開できます
- 回答は自動で保存されるので、途中でアプリを閉じても大丈夫です

### 2. 面接画面（全8問）

AIキャラクター「翔平」が音声で質問します。

質問を聞いたあと、**2つの方法**で回答できます。

- **声で回答する場合**: 画面中央の**「話す」ボタン**を押して話す → 終わったら**もう一度押して停止**
- **文字で回答する場合**: 画面の**「テキストで入力する」**を押して、キーボードで入力

回答できたら**「次へ」**を押して次の質問に進みます。

- **「戻る」**で前の質問に戻れます
- 左上の**「中断する」**でトップに戻れます（回答は自動保存されます）

### 3. 回答確認・提出画面

8問すべて回答すると、確認画面になります。

1. 回答内容を確認します
2. **ニックネーム**を入力します（本名でなくてOKです）
3. **電話番号 または LINE ID** を入力します
4. **「提出する」ボタン**を押して完了です

### 4. 完了画面

提出後、翔平からお礼のメッセージが流れます。

# 運営向け（採用担当の方の画面）

採用担当者が応募者の回答を確認するための画面です。

## 管理画面の開き方

スマートフォンまたはパソコンで https://ai-mendan.vercel.app/admin を開きます。

## 画面の見方

### 応募者一覧

管理画面を開くと、応募者の一覧が表示されます。

| 表示項目 | 説明 |
|------|------|
| 名前 | 応募者が入力したニックネーム |
| 連絡先 | 電話番号またはLINE ID |
| 日時 | 面接を提出した日付と時刻 |
| 音声 | 本番では録音の再生ができるようになります（デモ版ではアイコンのみ） |
| 判定 | 「推奨」「判定済」「未判定」のいずれかが表示されます |

**応募者の行をタップすると、詳細画面に移動します。**

### 応募者の詳細画面

詳細画面では、以下の2つが確認できます。

**AIレポート（応募者カルテ）**

「AIレポートを生成」ボタンを押すと、AIが回答内容を読み取って、以下の情報をまとめたレポートを自動で作成します。

- その人の価値観や人柄
- どんな働き方を望んでいるか
- 面接から読み取れる強み
- 向いていそうな部署や役割
- 二次面接で確認した方がよいこと
- 採用についてのAIのコメント

**回答内容**

8問それぞれの質問と、応募者の回答がそのまま表示されます。

### サンプルデータについて

初めて管理画面を開いたときは「たろう（サンプル）」というサンプルデータが表示されます。
実際に面接を行うと、新しいデータが一覧に追加されます。

# デモ版をお試しいただく際のご注意

## データの保存先について（重要）

デモ版では、面接の回答データは**お使いの端末（スマホやPC）の中だけ**に保存されます。

- 応募者として面接を受けたスマホと、**同じスマホで管理画面を開く**と回答が確認できます
- **別のスマホやPCからは、そのデータは見えません**
- ブラウザの履歴やデータを消去すると、回答データも消えます

> **お試しの手順**: まず面接を体験 → そのまま同じスマホで管理画面（/admin）を開いて確認

本番では、会社のサーバーにデータを保存するため、どの端末からでも管理画面で確認できるようになります。

# 本番で追加・改善する予定の機能

今回のデモは「体験いただくための検証版」です。本番環境では以下を対応いたします。

| 項目 | デモ版（今回） | 本番版（今後） |
|------|--------|--------|
| データの保存 | 各スマホの中だけ | 会社のサーバーに一括保存 |
| 管理画面のログイン | なし（誰でもアクセス可） | パスワードでログイン |
| 合否判定 | AIレポートのみ | 合格・不合格の管理機能 |
| 応募があった時の通知 | なし | メールで自動通知 |
| 合否連絡 | 手動で電話/メール | メールで自動送信も可能 |
| 面接時の音声 | 保存されない | サーバーに保存し、あとから再生可能 |
| 画面のデザイン | シンプルな構成 | 御社のイメージに合わせたデザイン |
| マイナビ・ことナビとの連携 | なし | 募集ページからそのまま面接へ |
| セキュリティ | デモ用の簡易な構成 | 通信の暗号化、データの保護、アクセス制限 |

## デザインについて

デモ版ではシンプルな構成としていますが、本番では「やさしい丸みのあるデザイン」「地域共生社会の世界観」に合わせたデザインに仕上げます。管理画面についても、実際の業務で使いやすいデザインを改めてご提案いたします。
